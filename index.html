⚡ Quick Setup

Get Claude Code running in under 3 minutes. Choose your device:

🪟 Windows Setup

  1. Install Claude Code CLI: 
    C:\> npm install -g @anthropic-ai/claude-code
  2. Get FREE API key:

    👉 Register: agentrouter.org/register?aff=TAGp
    🔑 Token: console/token

  3. Set environment variables: 
    setx ANTHROPIC_BASE_URL "https://agentrouter.org/"
setx ANTHROPIC_AUTH_TOKEN "sk-YOUR_TOKEN"
setx ANTHROPIC_API_KEY "sk-YOUR_TOKEN"
setx ANTHROPIC_MODEL "claude-sonnet-4-5-20250514"

    Restart terminal after setx!

  4. Launch: 
    C:\> cd your-project && claude

🍎 macOS Setup

  1. Install Claude Code: 
    $ npm install -g @anthropic-ai/claude-code
  2. Add to ~/.zshrc: 
    export ANTHROPIC_BASE_URL="https://agentrouter.org/"
export ANTHROPIC_AUTH_TOKEN="sk-YOUR_TOKEN"
export ANTHROPIC_API_KEY="sk-YOUR_TOKEN"
export ANTHROPIC_MODEL="claude-sonnet-4-5-20250514"
  3. Apply & Run: 
    $ source ~/.zshrc && claude

🐧 Linux Setup

  1. Install dependencies: 
    $ sudo apt update && sudo apt install nodejs npm git -y
  2. Install Claude Code: 
    $ npm install -g @anthropic-ai/claude-code
  3. Set env vars in ~/.bashrc: 
    export ANTHROPIC_BASE_URL="https://agentrouter.org/"
export ANTHROPIC_AUTH_TOKEN="sk-YOUR_TOKEN"
export ANTHROPIC_API_KEY="sk-YOUR_TOKEN"
  4. Run: 
    $ source ~/.bashrc && claude

📱 Android via Termux

Important: Install Termux from F-Droid (NOT Play Store — outdated version).

  1. Update & Upgrade: 
    $ pkg update && pkg upgrade
  2. Install Dependencies: 
    $ pkg install nodejs git nano -y
  3. Verify Versions: 
    $ node -v
$ npm -v
  4. Install Claude Code (Specific Version): 
    $ npm install -g @anthropic-ai/claude-code@2.1.112
  5. Run Installer Script: 
    $ node $PREFIX/lib/node_modules/@anthropic-ai/claude-code/install.cjs
  6. Set Environment Variables: 
    export ANTHROPIC_BASE_URL=https://agentrouter.org/
export ANTHROPIC_AUTH_TOKEN=sk-Z6Bxxxxxxxxxxxxxxxx
export ANTHROPIC_MODEL=claude-opus-4-6
export CLAUDE_CODE_USE_AUTH_TOKEN=true

    Add these to ~/.bashrc for persistence!

  7. 🚀 Run Claude: 
    $ claude

Pro Tips for Termux

  • • Use nano ~/.bashrc to make env vars permanent
  • • Run termux-wake-lock to prevent sleep during long tasks
  • • Use claude-sonnet-4-5-20250514 for best free-tier performance
  • • Keep Termux updated: pkg upgrade weekly
